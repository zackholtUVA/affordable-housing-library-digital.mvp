# Affordable Housing Library Digital MVP

Front-end-only MVP prototype built with Next.js, TypeScript, and Tailwind CSS.

## Design and Product Constraints
- Inter typography
- Dark mode default with light mode toggle
- High-end simple professional interface
- Placeholder-only domain content
- No backend, auth, API integrations, or personalization in MVP

## Routes
- `/` Home
- `/explore` Browse and filter options
- `/options/[slug]` Option detail template
- `/compare` Side-by-side compare
- `/next-steps` Action-oriented guidance
- `/basics` Glossary and terminology support

## Local Development
```bash
npm install
npm run dev
```

## Quality Commands
```bash
npm run lint
npm run test
npm run build
```
